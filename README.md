# Centre d'Affaires -- Centirio

Page vitrine simple en `index.html` pour présenter l'offre de domiciliation de Centirio.

## Contenu

- Offre de domiciliation pour comptables agréés
- Tarifs par durée
- Adresse du centre
- Emplacement Google Maps
- Boutons d'appel et WhatsApp

## Utilisation

Ouvrez simplement `index.html` dans un navigateur.

## Personnalisation

- Modifiez les tarifs dans le bloc `Domiciliation`
- Remplacez les coordonnées si besoin
- Ajustez les couleurs et les textes directement dans `index.html`
